# Sales Data Analysis Dashboard (Excel)

## Project Overview

This project presents an interactive **Sales Analysis Dashboard** built in **Microsoft Excel** to analyze sales performance, profit, and customer segments.
The dashboard provides clear business insights through visualizations, KPIs, and interactive filters

## Key KPIs

* **Total Sales:** $2,297,201
* **Total Profit:** $286,397
* **Profit Margin:** 12%
* **Total Quantity Sold:** 37,873

## Dashboard Insights

The dashboard analyzes sales using multiple perspectives:

* **Sales by City** – Identify top-performing cities
* **Sales by Region** – Compare regional sales performance
* **Sales by Segment** – Consumer, Corporate, and Home Office
* **Top 10 Products by Sales**
* **Category-wise Quantity Distribution**
* **Shipping Mode Analysis**
* **Monthly Sales Trends**
* **Daily Sales Trend Analysis**

## Interactive Filters

Users can dynamically filter the dashboard using:

* **Region**
* **Year**
* **Month**

These slicers allow deeper analysis and help in identifying trends and patterns quickly.

## Tools & Techniques Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning
* KPI Calculations
* Interactive Dashboard Design

## Business Value

This dashboard helps businesses to:

* Track overall sales performance
* Identify top-selling products
* Analyze regional sales trends
* Monitor monthly and daily sales patterns
* Support data-driven decision making
